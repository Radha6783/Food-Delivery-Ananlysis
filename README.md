# Food-Delivery-Ananlysis
 Project Overview
     This project analyzes a food delivery dataset to extract meaningful insights about customer orders, restaurant performance, revenue trends, and user behavior. The analysis was done using Python and Pandas in       Google Colab.
Objectives
   Analyze total orders and revenue
   Compare Gold vs Regular members
   Identify top-performing restaurants
   Study ratings and cuisine impact on revenue
   Find city-wise and quarter-wise trends
Tools & Technologies
  Python
  Pandas
  NumPy
  Google Colab
  GitHub
Dataset Used
   orders.csv – order details
   users.json – user information
   restaurants.csv – restaurant data including cuisine & ratings
Key Insights
   Gold members contributed significantly to revenue
   Highly rated restaurants generated more revenue
   Certain cuisines performed better than others
  Revenue trends varied by quarter and city
What i learned
   Data cleaning and preprocessing
   Merging datasets using Pandas
   Groupby operations & aggregations
  Extracting insights from real-world data
Conclusion
   This project improved my data analysis skills and helped me understand how data-driven decisions can be made in the food delivery industry.
